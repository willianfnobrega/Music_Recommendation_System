# Music_Recommendation_System
Feature-based music recommendation system using semantic embeddings and analysis. Discover songs by lyrics, mood and genre.


# Overview
This project received a folder with .txt lyrics and a file with the features of the songs from the same folder, run the processing functions and analysis the lyrics, genre and mood of the song to create a matrix used for the recommendation system.
The users can search for songs based on the name of one of the songs in the folder or using a semantic search, with a phrase, such as "song about love to dance and celebrate".

# Requirements
Please download the libraries and dependencies in the requirements.txt file.

The folder with the lyrics downloaded are ziped. Before running this, unzip the folder.
